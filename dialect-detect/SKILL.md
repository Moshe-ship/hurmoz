# اكتشاف اللهجة العربية

## اللهجات الرئيسية وعلاماتها

### مصري (EGY)
علامات مميزة:
- "إزيك"، "كده"، "عايز"، "إيه"، "فين"، "ليه"
- "بتاع"، "حاجة"، "خالص"
- الجيم تنطق G: "جميل" = "gamiil"

أمثلة للاكتشاف:
```
المدخل: "إزيك يا صاحبي، عايز أروح السينما النهاردة"
اللهجة: مصري (confidence: 0.96)
العلامات: ["إزيك", "عايز", "النهاردة"]
```

```
المدخل: "الأكل ده جامد خالص والله"
اللهجة: مصري (confidence: 0.92)
العلامات: ["ده", "جامد", "خالص"]
```

```
المدخل: "مفيش حاجة تقلق منها، كله تمام"
اللهجة: مصري (confidence: 0.89)
العلامات: ["مفيش", "حاجة", "تمام"]
```

## التمييز بين اللهجات المتشابهة

بعض الكلمات مشتركة بين لهجات. قواعد التمييز:

| الكلمة | مشتركة بين | كيف تفرّق |
|--------|-----------|-----------|
| "شلونك" | خليجي + عراقي | عراقي يستخدم "شكو ماكو"، خليجي يستخدم "وش" |
| "كيفك" | شامي + فصحى | شامي يتبعها "شو"، فصحى تتبعها "كيف حالك" |
| "أبي" | خليجي | إذا بمعنى "أريد" = خليجي، إذا بمعنى "والدي" = أي لهجة |
| "ليش" | خليجي + شامي | السياق يحدد — شامي يستخدم "هلق" و"بدي" |

## السلوك

1. اكتشف اللهجة من أول رسالة
2. ردّ بنفس اللهجة
3. إذا ما تعرف اللهجة — استخدم عربي بسيط مفهوم
4. ما تخلط لهجات — إذا المستخدم مصري ما ترد بخليجي
5. إذا غيّر المستخدم للإنجليزي — ردّ بالإنجليزي
6. إذا المستخدم يخلط عربي وإنجليزي — ردّ بنفس الأسلوب
7. إذا اللهجة غير واضحة من رسالة واحدة — انتظر رسالة ثانية قبل ما تثبّت اللهجة
